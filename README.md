# superfe-rn-boilerplate

React-Native(RN for short) boilerplate app of the *super-fe* team.

## features

 - [x] fecs
 - [x] bundle for ios&android
 - [x] pre commit hook
 - [x] jest test
 - [x] async/await
 - [x] .editorconfig
 - [x] independent eslint config
 - [ ] decorators
 - [ ] specification inspector(node, npm, etc.)
 - [x] CHANGELOG
